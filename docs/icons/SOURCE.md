# Icon Source

Favicon and app icons are generated from a black restaurant emblem with crossed cutlery, chef hat, and ribbon text.

Composition details:
- Base crossed fork/knife silhouette: `Fork & knife.svg` from Wikimedia Commons
  - URL: https://commons.wikimedia.org/wiki/File:Fork_%26_knife.svg
  - Direct file URL: https://commons.wikimedia.org/wiki/Special:FilePath/Fork%20%26%20knife.svg
- Visual style reference for hat+ribbon layout: user-provided reference image in this project conversation
- Final SVG composition: customized for Don Jose's with ribbon text `Don Jose's`

Generated files:
- `public/favicon.svg`
- `public/favicon.ico`
- `public/icons/icon.svg`
- `public/icons/icon-*.png`

Generation script:
- `scripts/generate-icons.sh`
